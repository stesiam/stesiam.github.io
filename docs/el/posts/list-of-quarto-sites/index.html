Εισαγωγή

Σημείωση

Αποφάσισα να έχω τη λίστα και σε μορφή project στην αντίσοιχη σελίδα στο GitHub Repo.

Τη στιγμή που επεξεργάζομαι το συγκεκριμένο κείμενο το Quarto έχει ξεπεράσει σε δημοφιλία άλλα πακέτα δημιουργίας ιστοσελίδων (3.4χιλ. αστέρια), συγκρινόμενο με πακέτα κατασκευής ιστοσελίδων, όπως το blogdown (1.7k ) και το distill (400 ). Δεδομένης της αυξημένης δημοφιλίας του Quarto, πήρα τη πρωτοβουλία να κάνω μία λίστα ιστοσελίδων που το χρησιμοποιούν. Με αυτό το τρόπο αναλυτές δεδομένων που είναι χρήστες άλλων μεθόδων κατασκευής ιστοσελίδων (e.g., distill, blogdown) θα έχουν την ευκαιρία να δουν τις δυνατότητες μιας ιστοσελίδας Quarto. Επιπλέον, αυτή η λίστα ενδεχομένως να βοηθήσει και όσους έχουν ήδη προσθέσει το Quarto στην εργαλειοθήκη τους μιας και θα μπορούν να παίρνουν ιδέες από τους υπόλοιπους από μία συγκεντρωτική λίστα.

Δυστυχώς, όπως πιθανότατα είναι ήδη προφανές, αυτή η λίστα δεν είναι δυνατό να περιλαμβάνει κάθε ιστοσελίδα Quarto. Ως επί το πλείστον είναι σελίδες που έχω δει και μου άρεσαν. Αν έχετε υπόψιν σας κάποια άλλη σελίδα ή ακόμα και τη δική σας σελίδα, μπορείτε να γράψετε ένα σχόλιο ή να μου στείλετε μήνυμα και θα την προσθέσω το συντομότερο δυνατόν.

Έμπνευση

Λίγα χρόνια νωρίτερα αποφάσισα να φτιάξω μία ιστοσελίδα προκειμένου να έχω ένα σημείο να δείχνω τα roject και όχι σε διάσπαρτα repos στο GitHub. Έτσι δοκίμασα αρκετές λύσεις οι οποίες θα έπρεπε να υποστηρίζουν τα Rmd αρχεία (R Markdown αρχεία). Εν τέλει είχα καταλήξει να χρησιμοποιώ το Distill, και ειλικρινά το λάτρεψα, καθώς η ιστοσελίδα συνδύαζε την απλότητα και στοιχεία που την έκαναν να φαίνεται αρκετά επαγγελματική. Παρά τα όποια προβλήματα υπήρχαν με το συγκεκριμένο (πολλά εκ των οποίων λύθηκαν με το Quarto), ήμουν ιδιαίτερα ικανοποιημένος. Ωστόσο, εκείνη τη περίοδο ήμουν ένας αρχάριος προγραμματιστής με αρκετά περιορισμένη γνώση HTML και CSS. Αυτό σήμαινε ότι θα μπορούσα να κάνω λίγες αλλαγές και θα μου έπαιρναν πάρα πολύ χρόνο για να υλοποιήσω τις όποιες στυλιστικές αλλαγές. Την ίδια περίοδο ανακάλυψα μία ιστοσελίδα που λίστα με ιστοσελίδες Distill. Με αυτή τη λίστα είχα καταφέρει:

  • Κατανοήσω τους περιορισμούς του Distill
  • Κίνητρο να ξεκινήσεις παίρνοντας έπνευση από τις ιστοσελίδες άλλων
  • Εύκολη πρόσβαση σε ενδιαφέροντα χαρακτηριστικά σελίδων συνοδευόμενα από την αντίστοιχη σελίδα κώδικα

Το 2022, ανακοινώθηκε το Quarto και τα δεδομένα συγγραφής και δημοσίευσης άρθρων, αναφορών, ιστοσελίδων άλλαξαν αρκετά. Αρκετοί χρήστες του Distill αποφάσισαν να κάνουν την αλλαγή και χρησιμοποιούν ήδη το Quarto για την προσωπική τους ιστοσελίδα (με βάση τη λίστα ιστοσελίδων Distill). Κάποιοι ακόμα μπορεί να διστάζουν για αυτή την αλλαγή μιας και το Quarto είναι αρκετά νέο (και μπορεί να υπάρχουν σημαντικά θέματα που να μην έχουν επιλυθεί). Ένα ακόμα στοιχείο είναι η ώρα που απαιτείται για να μάθουμε να χειριζόμαστε μιας και η τεκμηρίωσή του είναι ιδιαίτερα εκτεταμένη. Αυτός ήταν κα ο λόγος άλλωστε που πιστεύω ότι αυτή η λίστα να είναι σημαντική, πορκειμένου να καταστήσει αυτόν τον νέο τρόπο πιο προσβάσιμο στους νέους χρήστες και να ενθαρρύνει την υιοθέτηση νέων λειτουργιών από τους υδιστάμενους χρήστες.

Κατάλογος ιστοσελίδων Quarto

Μία ενδεικτική λίστα των ιστοσελίδων που έχουν κατασκευαστεί χρησιμοποιώντας το Quarto είναι η εξής:

Πολύγλωσσες ιστοσελίδες

Δυστυχώς, το Quarto δεν υποστηρίζει από μόνο του τη δημιουργία ιστοσελίδων που να υποστηρίζουν διαφορετικές γλώσσες. Αυτό δημιουργεί ενδεχομένως πρόβλημα σε όσους επιθυμούν να προωθήσουν το περιεχόμενό τους και στη μητρική τους γλώσσα, όπου μπορεί να διαφέρει από τη αγγλική. Προς το παρόν η λύση που προσφέρει επίσημα το Quarto είναι μέσω των project profiles, αλλά η εφαρμογή του μου φαίνεται αρκετά περίπλοκη και με μικρή αναφορά σε όποια τεκμηρίωση (documentation) ως προς τη συγκεκριμένη χρήση. Αν σας ενδιαφέρουν οι εξελίξεις σχετικά με την υποστήριξη πολλών γλωσσών από το Quarto μπορείτε να δείτε τη συζήτηση για το συγκεκριμένο θέμα στη σελίδα του project στο GitHub. Προς το παρόν, επίσημα, το Quarto δεν υποστηρίζει με έναν ξεκάθαρο τρόπο πολύγλωσσες ιστοσελίδες, ηλεκτρονικά βιβλία κοκ.

Παρόλο τη μη επίσημη υποστήριξη του Quarto, δημιουργήθηκε το babelquarto, ένα R πακέτο το οποίο προσπαθεί να λύσει το πρόβλημα και να κάνει τη δημιουργία πολύγλωσσων σελίδων δυνατή και εύκολη. Υπάρχουν κάποιες ιστοσελίδες που έχουν ήδη εντάξει το babelquarto στην εργαλειοθήκη τους και πλέον έχουν ιστοσελίδες σε διαφορετικές γλώσσες. Μία εκ των οποίων και αυτή η ιστοσελίδα που διαβάζετε αυτή τη στιγμή. Ένα άλλο παράδειγμα πολύγλωσσης ιστοσελίδας (ιαπωνικά / αγγλικά) αποτελεί η ιστοσελίδα του Joel Nitta.

Ελπίζω όλα αυτά να σας φάνηκαν ενδιαφέροντα και γιατί όχι να σας έπεισα να φτιάξετε και εσείς τη δική σας ιστοσελίδα σε Quarto.
