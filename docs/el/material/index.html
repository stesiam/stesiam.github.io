Υλικό

Στο πλαίσιο των σπουδών μου, ως προπτυχιακός φοιτητής, έχει χρειαστεί να κάνω αρκετές εργασίες. Με βάση αυτό αποφάσισα να αναρτήσω ορισμένες από αυτές. Επιπλέον, αναρτώ και υλικό που θα βοηθήσει τους υπόλοιπους συμφοιτητές μου που για διάφορους λόγους (π.χ. υγείας, οικονομικούς κτλ.) δεν μπορούν να ολοκληρώσουν τις σπουδές τους.
