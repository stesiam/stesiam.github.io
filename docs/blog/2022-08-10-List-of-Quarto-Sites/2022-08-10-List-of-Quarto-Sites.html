Introduction

Given the increased popularity of Quarto, I took the initiative to make a list of some sites that are using this framework. That way, users of other popular frameworks (e.g., distill, blogdown) will have the chance to see the features of a Quarto site. Furthermore, existing Quarto users can take some inspiration from other web pages.

Unfortunately, it is certain, that this list does not contain all the Quarto pages. Although, you are welcome to edit the page (to add yours) or comment on this article.
